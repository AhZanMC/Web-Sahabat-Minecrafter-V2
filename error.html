Maaf Halaman Tersebut Tidak Bisa Diakses

  • Website yang kamu datanagin eror,penyebabnya adalah :

    1. Website sedang mengalami maintance/di matikan
    2. Website yang kamu tuju memang nggak ada
    3. Koneksi internet yang kurang bagus
    4. Keknya kuota internetmu habis :V
    5. Mungkin fitur yang kamu akses masih beta/cuma pajangan :V
    6. Mungkin tergantung ama amal ibadah :V

  • Solusi :

    1. Cek koneksi internet kamu,pastikan jaringan internet bagus
    2. Pastikan kuota internetmu ada,kalo nggak ada bisa nebeng wifi tetangga :V
    3. Pastikan kuota internetmu aman/masih ada
    4. Cek Proxy dan Firewall
    5. Jalankan Windows Network Diagnostics
    6. Laporkan ke admin/owner
